Программы

  • VS Code

    Мощный редактор кода с поддержкой множества языков, расширяемыми плагинами и интеграцией с системами контроля версий.

    Интеллектуальная IDE для JavaScript и веб-разработки, предлагающая мощные инструменты для отладки, тестирования и рефакторинга кода.

  • Figma

    Современный инструмент для дизайна интерфейсов и прототипирования, позволяющий командам совместно работать в реальном времени.

  • Photoshop

    Графический редактор, предоставляющий обширные возможности для редактирования изображений, создания композиций и цифрового искусства.

Hard skills

  • HTML5

    Современный язык разметки для структурирования веб-страниц с поддержкой мультимедийных и семантических элементов.

  • CSS3

    Стиль и оформление сайтов с использованием новых возможностей для анимаций, трансформаций и адаптивного дизайна.

  • JavaScript

    Язык программирования для динамичного взаимодействия на веб-страницах, создания интерактивных элементов и логики.

  • NodeJS

    Платформа для выполнения JavaScript на сервере, обеспечивающая быстрое создание масштабируемых сетевых приложений.

  • GitHub

    Онлайн-платформа для управления версиями кода с совместной работой и контроля изменений в проектах.

  • БЭМ

    Методология написания CSS со структурным подходом, улучшающая поддержку, масштабирование и переиспользование стилей.

  • ООП

    Объектно-ориентированное программирование (Парадигма программирования) — подход, использующий объекты и классы для повышения читаемости и поддержки кода.

  • Кроссбраузерность

    Обеспечение корректного отображения и работы сайта во всех популярных браузерах и их версиях.

  • Адаптивность

    Создание интерфейсов, автоматически подстраивающихся под разные размеры экранов и устройства для комфортного пользования.
